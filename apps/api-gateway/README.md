# API Gateway

Entry point for all client requests. Routes traffic to appropriate microservices.

## Features

- Request routing
- Rate limiting
- CORS handling
- Security headers (Helmet)
- Error handling
- Health checks

## Environment Variables

See `.env.example` in root directory.

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Test

```bash
pnpm test
```
